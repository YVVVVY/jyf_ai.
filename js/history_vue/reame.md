# js 历史

- const
  常量， es5没有专门的常量声明
  简单数据类型：值不能改变
  对象呢？ 值可以改变，指向的内存地址不能改变。

- 赋值
  简单数据类型 值传递
  复杂数据类型 地址传递（引用式赋值）
  const 类型不同处理不同

  内存栈 连续，空间小 快
  内存堆 不连续 空间比较大 慢
  const 简单数据类型，复杂数据类型虽然表现不一致，内存栈中的值不会发生改变

es6 第一个新特性 let const ，弥补了es5只有var的不足，支持块级作用域，支持常量。
早期js 用来设计页面交互，简单， es6要企业级别的大型应用开发语言，拥抱其他语言开发者和java，c++一样，不再是之前的老破小 
  - TDZ 变量提升 代码的阅读和执行的歧义（可读性）
  - 块级作用域
